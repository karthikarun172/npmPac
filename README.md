# what is this?

Get a perfect shad

# Installation

`npm i shads --save`

Then...

```
    import {Shad} from "shads"

    Shad({
        shadow_type:'soft',
        padding:false
    })
```

# Options

Shadowizard supports 2 options,both of which are optional

- _shadow_type_ - _hard | soft_ (Default soft)
- _padding_ - _boolean_ (Default to false)
